# funcoes-node.js-PW2
Projeto desenvolvido para praticar conceitos básicos de JavaScript utilizando Node.js

## Conteúdos abordados
- Estruturas condicionais
- Laços de repetição
- Operações matemáticas
- Operadores lógicos e relacionais
- Tipos de variáveis

## Como executar
No terminal, execute:
node index.js
